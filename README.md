## 2、.md格式
官网：https://markdown.com.cn/basic-syntax/  

（1）标题：#  

（2）强调：*斜体*  

（3）代码：
    1、
    ``  
    2、
    ```js
    ```
（4）换行：在一行的末尾添加两个或多个空格，然后按回车键  

（5）加粗：**bold**

（5）链接：[显示名](url)  

（6）图片：![图片alt](url "图片title")
